# DQA 同步工具 - 使用帮助
## 1. 功能说明
- 原始功能：从Excel创建禅道工单
- 服务功能：开启/关闭同步服务（ngrok+Flask）
- 系统设置：修改.config配置文件
- 禅道创建：手动添加Jira记录并批量创建禅道工单

## 2. 操作步骤
### 2.1 JQL查询
1. 在主界面输入JQL语句，点击「列表查询」获取Jira数据
2. 支持的JQL格式：`key=CNUXBTRACK-123` 或 `project=CNUXBTRACK`

### 2.2 禅道创建
1. 点击「添加记录」打开禅道创建弹窗
2. 选择Jira项目、输入编号，自动查询Jira信息
3. 选择禅道模块，点击「添加」将记录加入表格
4. 点击「确认提交」批量创建禅道工单

### 2.3 服务管理
- 点击「开启同步服务」启动ngrok和Flask服务
- 服务状态会实时显示在主界面底部
- 点击「关闭同步服务」停止所有服务

## 3. 常见问题
### 3.1 JQL查询失败
- 检查JQL语句格式是否正确
- 确认Jira项目名称和编号是否存在
- 检查网络连接和Jira权限

### 3.2 禅道创建失败
- 检查.config文件中的禅道模块映射配置
- 确认禅道指派人ID和模块ID是否正确
- 检查禅道接口权限
